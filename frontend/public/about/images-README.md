# About page images

Place images here for the About page. Paths are relative to `public/`, so use these filenames:

- **about-hdrwa.jpg** — About HDRWA section (right side). Use a real healthcare/research scene (e.g. field work, clinic, data collection).
- **card-why-share.jpg** — Card 1 “Why Share Your Data?”. Use a lab or research collaboration image.
- **card-management.jpg** — Card 3 “Data Management”. Use a dashboard/tech interface or data visualization image.

Card 2 (Data Privacy, Security and Ethics) uses an inline illustration; no image file needed.

Use the same aspect ratio (e.g. landscape 16:10 or 4:3) and consistent tone for a uniform look.
